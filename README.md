# Promote App Landing Page

## Project Overview

The **Promote App Landing Page** is a responsive website built using **HTML, CSS, and JavaScript** based on the provided Figma design.
The landing page showcases a platform that connects **businesses, influencers, and celebrities** to collaborate on marketing campaigns.

This project demonstrates the ability to convert a **UI design into a functional web page** while maintaining layout structure, styling consistency, and user-friendly presentation.

---

## Technologies Used

* HTML5
* CSS3
* JavaScript
* Google Fonts (Poppins)

### 2. How Promote App Works

Explains how the platform functions for both businesses and influencers.

#### For Businesses

* Register your business on Promote App
* Choose influencers based on target audience
* Launch promotion campaigns

#### For Influencers

* Register as an influencer
* Complete your profile verification
* Receive collaboration requests from brands
* Promote and earn through campaigns

## How to Run the Project

1. Download or copy the project folder.
2. Ensure the folder contains:

   * `index.html`
   * `style.css`
   * `script.js`
3. Open the **index.html** file in any web browser.

The landing page will load locally.
